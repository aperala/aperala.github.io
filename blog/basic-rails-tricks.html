Things to remember in a basic Rails CRUD App

Generating Models with References

$ rails generate model reply post:references

The command above generates a migration with a post_id column (and an index on this field).

Adding a Users Controller and Views when Using Devise

In addition to generating a Users Controller, add resources for your desired routes (make sure it's underneath the devise_for line): 

        #routes.rb
        devise_for :users
        resources :users, :only => [:show]
